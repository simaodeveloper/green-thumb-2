# GreenThumb

A plant simulator make with ES6, SASS, Handlebars, Webpack and <3

## Como rodar este projeto

##### Rodando o modo de desenvolvimento


```
  yarn run watch:dev
```

##### Gerando o pacote para deploy

```
  yarn run build:dist
```

## Link da Aplicação

https://greenthumb2.surge.sh
